# AxDatePicker [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

An AxDatePicker component displays a control that allows the user to select a date value. It is typically used for non-bound fields.
