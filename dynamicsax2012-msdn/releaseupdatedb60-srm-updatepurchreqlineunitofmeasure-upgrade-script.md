## Remarks

The string PurchUnit and ProjSalesUnitId fields have been renamed to “DEL\_PurchUnit”, “DEL\_ ProjSalesUnitId” respectively, replaced with RefRecId fields with the PurchUnitOfMeasure, ProjSalesUnitOfMeasure.
