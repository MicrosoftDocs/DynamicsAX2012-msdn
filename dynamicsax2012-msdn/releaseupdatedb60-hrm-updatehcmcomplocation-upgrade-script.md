## Remarks

This upgrade will discard duplicate records if the LocationId and Description field in the HRMCompLocation table are identical. The new unique identifier CompLocationId field in the HcmCompLocation table combines the LocationId and DataAreaId field in the HRMCompLocation table from the first record found. All records found that are identified as duplicates in other companies will be discarded. If only one company exists, the DataAreaId field will not be appended.
